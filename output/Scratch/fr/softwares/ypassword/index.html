YPassword

Une gestion simple, sécurisée et portable de ses mots de passes web.

Souvenez vous d’un seul mot de passe de bonne qualité, le reste suis.

Ici vous trouverez :

  • Un widget ;
  • un Applescript pour simuler la copie dans les champs textes protégés ;
  • une application web écrite avec Cappuccino,
  • une application web pour iPhone ;
  • un script utilisable en ligne de commande.

Dans peu de temps je créerai une application iPhone pour YPassword.
